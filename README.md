# FIVEM ENITY–IDENTITY

DISCORD:
discord.gg/enity


## Funkcje

- Tworzenie tożsamości postaci z danymi osobowymi
- Wybór płci (mężczyzna/kobieta)
- Ustawienie daty urodzenia postaci
- Wybór wzrostu
- Wybór narodowości
- Miejsce urodzenia
- Nowoczesny interfejs użytkownika
- Wielokrokowy formularz z podsumowaniem

## Wymagania

- ESX Legacy
- oxmysql

## Konfiguracja

Edytuj plik `config.lua` aby dostosować:
- Minimalny/maksymalny wiek postaci
- Zakres wzrostu
- Dostępne narodowości
